File Transfer Progress

		{
		  "stat":"rprg",
		  "username": "username",
		  "ip":"IP",
		  "port":"port",
		  "dir":"fileDir",
		  "total": "fileSize as byte",
		  "current": "currently downloaded as byte",
		  "speed": "transfer speed as kilobyte second",
		 }

	sending progress (upload progress)
		{
		  "stat":"sprg",
		  "username": "username",
		  "ip":"IP",
		  "port":"port",
		  "dir":"fileDir",
		  "total": "fileSize as byte",
		  "current": "currently upload as byte",

Message Protocol

	A message has receive
		{
		  "stat": "rmsg"
		  "person": "someone",
		  "content": "Message content",
		  "contentType": "text"
		}

	You send a message
		{
		  "stat": "smsg"
		  "person": "you",
		  "content": "Message content",
		  "contentType": "text"
		}

	fail to send your message
		{
		  "stat": "fmsg"
		  "person": "you",
		  "content": "Your faild message content",
		  "contentType": "text"
		}


*all size parameters units are byte.
*download and upload speed units kilobyte per second.
